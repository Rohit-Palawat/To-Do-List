# 📝 Task Flow - simplify your daily life

A full-stack MERN To-Do List application that helps users organize and manage daily tasks efficiently.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Responsive user interface
- RESTful API integration
- MongoDB database storage

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

## 📂 Project Structure

```
To-Do-List/
├── frontend/
├── backend/
└── README.md
```

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/Rohit-Palawat/To-Do-List.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
